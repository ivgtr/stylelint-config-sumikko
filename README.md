#### これは何?
Stylelint + Prettier のルールセット  
[eslint-config-sumikko](https://github.com/ivgtr/eslint-config-sumikko)と併用してください

## Install

```
npm install -D stylelint-config-sumikko stylelint
```

## Usage

`.stylelintrc`
### JavaScript

```json
{
  "extends": ["stylelint-config-sumikko"]
}
```
